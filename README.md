# Spark&Heal - Life After Breast Cancer

This is the static website for **Spark&Heal**, a community-driven platform for breast cancer survivors.

## Hosted at
[https://thewallinna.github.io/Innashchepanska/](https://thewallinna.github.io/Innashchepanska/)

## Mission
Spark&Heal acts as a "virtual bridge" between traditional healthcare and post-treatment recovery. Founded by **Inna Shchepanska**, it provides evidence-based education, shared survivor stories, and tools to help women improve their quality of life during remission.
